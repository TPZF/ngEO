2. Interface Overview

The user interface home page is the following: It is composed of :

  • a header containing a menu bar. The entries :
    • About: gives a quick introduction to ngEO
    • Data Services Area: is the main menu to deal with catalog searches and data access services. At start up, the Data Services Area entry is activated. An active menu entry is colored in blue.
    • My account: is the menu for monitoring download managers, data access requests and user preferences.
    • User Guide: is a link to the online user guide launched from the application.
  • a central area which is dedicated to the map displaying. At start up, the map is the 2D map.

  • a footer which displays the currently selected dataset. In the beginning, no dataset is selected.
